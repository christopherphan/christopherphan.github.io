Implicit differentiation

Javascript must be enabled for this page to work.

Below we have a plot of the Folium of Descartes, the curve defined by

x3 + y3 = 3xy

By implicit differentation, we found that

dy/dx = (y - x2)/(y2 - x).

Consider the point (x, y), which is on the Folium.
